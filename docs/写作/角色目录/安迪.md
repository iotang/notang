# 安迪

安迪（Andy）是一位皮卡丘。曾用名有“一只火龙皮卡丘”。

## 人物特点

安迪是一个四处旅行（摸鱼）的吟游诗人。

他是一个理想主义者，他对那些非正义的行为极其不爽。但是，他也是一只非常不靠谱的宝可梦，你千万不要指望他能帮到你什么忙。

他常常带着一把吉他或者尤克里里，出现在酒馆里面，而且他通常会点上一杯红茶。

他的理想生活是能够天天摸鱼。
