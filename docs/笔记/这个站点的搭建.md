# 这个站点的搭建

这个站点是使用 MkDocs 搭起来的。

博客什么的，自然有像 Hexo、WordPress 等强大的平台。不过强大和简单不可兼得，那些平台对于我来说还是太花里胡哨了，而 MkDocs 却正合适。

毕竟这个站点也不算是个博客啊，其实是个百科网站。博客什么的，以后再说吧。

## 安装和使用

去看看 [MkDocs 官方文档](https://www.mkdocs.org/)。

可以使用很棒的 Material for MkDocs 主题：[Material for MkDocs 官方文档](https://squidfunk.github.io/mkdocs-material/)。

也有中文教程：[shafish.cn](https://shafish.cn/blog/mkdocs/)。

## 部署到 Github 上

去看看 [Material for MkDocs 官方文档](https://squidfunk.github.io/mkdocs-material/publishing-your-site/)。
